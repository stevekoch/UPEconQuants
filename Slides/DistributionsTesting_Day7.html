The simple hypothesis

  1. Compare one distribution to a known distribution
  2. Typical..
    1. \(H_0\): data from some known distribution
    2. \(H_a\): data is not from that distribution
  3. Structure of Test
    1. Take ECDF of your data and plot
    2. Specify the “known” distribution and Plot
    3. Measure biggest gap
    4. Compare to critical values (found in tables…)

Test statistics

  1. Two-tailed \[ D = \sup_x \left| F_0 (x) - \hat{F}(x) \right| \]

  2. One-tailed (not so common)

\[ D = \sup_x F_0 (x) - \hat{F}(x) \]

Two-sample version

  1. Assume \(x\) has sample size \(N\)
  2. Assume \(y\) has sample size \(M\)

\[ D_{n,m} = \sup_x \left| \hat{F}_{n,x} - \hat{G}_{m,y} \right| \]

Critical values

  1. The reject criteria \[ D_{n,m} > c(\alpha)\sqrt{\frac{N+M}{NM}} \]

  2. For \(\alpha = 0.05\), \(c(\alpha) = 1.358\)

  3. Computer packages use…

\[ c(\alpha) = \sqrt{-\ln \left( \frac{\alpha}{2}\right) \cdot \frac{1}{2}} \]

Expected Utility Backdrop

Preferences or choices (in uncertain situations)

  1. Roughly this developed from Expected Utility
  2. Primary focus was based on the mean and variance
    1. Higher mean: good
    2. Lower variance: good
  3. But, expected utility is a function of all moments
  4. Therefore, would also need assumptions about all derivatives of utility

A more general framework

  1. Utilities \(u_a\), \(u_b\) over prospects \(a\), \(b\)
  2. Not worried if this is under certainty
  3. Uncertainty would assume expected utility
  4. Not sure how this carries over to Alex’s work
  5. Assume \(u_a<u_b\)
  6. Consider probabilities \(\alpha\) and \(\beta\) with \(\alpha < \beta\)

Independence axiom

\[ (1-\beta)u_a + \beta u_b > (1-\alpha)u_a + \alpha u_b \]

  1. \(u_b\) was preferred
  2. Higher probability on \(u_b\) preferred, too.

Extend to \(n\) prospects - equality

  1. Notational convenience: \(u_i>u_j\) if \(i>j\)
  2. Probabilities: \(\alpha_i\) and \(\beta_i\)
  3. Assume 0: \(\alpha_i=\beta_i\), \(\forall i\).

\[ \sum_{i=1}^n \beta_i u_i = \sum_{i=1}^n \alpha_i u_i \]

Extend to \(n\) prospects - preference

  1. Assume 1: \(\alpha_j>\beta_j\) and \(\alpha_k<\beta_k\)
  2. Assume 2: \(\alpha_i=\beta_i\) \(\forall i\), \(i\neq j,k\)

\[ \sum_{i=1}^n \beta_i u_i > \sum_{i=1}^n \alpha_i u_i \]

Rearranging with respect to the probabilities

  1. By assumption: \(\beta_i \leq \alpha_i\), \(i= \{1, 2, \dots, j, \dots, k-1\}\)
  2. Thus, because of \(j\) assumption:

\[ \sum_i^{k-1} \beta_i < \sum_i^{k-1} \alpha_i \]

Implications for the remainig probabilities

  1. By assumption: \(\beta_i=\alpha_i\), \(i=\{k+1, k+2, \dots, n\}\)
  2. Note: \(\alpha_j\), \(\alpha_k\) have to offset \(\beta_j\), \(\beta_k\)
  3. Probabilities must sum to one…

\[ \sum_{i=1}^r \beta_i = \sum_{i=1}^r \alpha_i, r=\{k, k+1, \dots, n \} \]

Putting this together

\[ \sum_{i=k}^r \beta_i \leq \sum_i \alpha_i, r=\{1, \dots, n \} \]

  1. Cumulative distribution of preferred prospect never exceeds that of the inferior prospect.
  2. This is independent of utility functional form (\(u'>0\), assumed)
  3. Hadar and Russell (1969)
    1. If \(P\) stochastically dominates \(P'\), \(P\) is preferred to \(P'\).
    2. If \(P\) is preferred to \(P'\) for all utility functions, \(P\) stochastically dominates \(P'\)
    3. Extending results of (quirksaposnik1962?)
  4. These have a welfare interpretation (Maasoumi and Racine 2016)

Stochastic Dominance

First order stochastic dominance (FSD)

  1. Assume \(u'>0\)
  2. The probability function \(g\) is said to be at least as large as the probability function \(f\) in the sense of first order stochastic dominance if and only if:

\[ G(x_i) \leq F(x_i) \qquad \forall x_i \in X \]

Why \(u'>0\)?

  1. Increasing utility
    1. Obvious: \(u(y) > u(x)\) if \(y>x\)
    2. FSD: prob\((y) >\) prob\((x)\), when \(y>x\)
    3. Reinforcing!
  2. Better outcomes are more likely
  3. So, anyone that prefers more will be better off with the FSD distribution

Second order stochastic dominance (SSD)

  1. Continue with \(u'>0\)
  2. Add further restriction: \(u'' \leq 0\)
  3. Risk aversion incorporated, now.
  4. The probability function \(g\) is said to be at least as large as the probability function \(f\) in the sense of second order stochastic dominance if and only if:

\[ \int_{x_1}^x G(y)dy \leq \int_{x_1}^x F(y)dy \qquad \forall x_i \in X \]

Third order stochastic dominance (TSD)

  1. Whitmore (1970)
  2. Assume: Risk premium associated with uncertainty becomes smaller with wealth
  3. Requires: \(u'>0\), \(u''\leq 0\) and \(u''' \geq 0\).
  4. The probability function \(g\) is said to be at least as large as the probability function \(f\) in the sense of if and only if:

\[ \begin{aligned} \int_a^x \int_b^y (F(z) - G(y))dz dy & \geq 0 \qquad \forall x , \text{ and} \\ \int_a^b (F(y)-G(y)) dy & \geq 0 \qquad \forall x \end{aligned} \]

Dominance Testing

Barrett and Donald (2003) notation1

\[ \begin{aligned} \mathcal{J}_1(z;G) &= G(z) \\ \mathcal{J}_2(z;G) &= \int_0^z G(t)dt = \int_0^z \mathcal{J}_1(t;G)dt \\ \mathcal{J}_3(z;G) &= \int_0^z \int_0^t G(s) ds dt = \int_0^z \mathcal{J}_2 (t;G) dt \end{aligned} \]

  1. \(SD^j\) implies \(SD^{j+1}\)

Davidson and Duclos (2000)

\[ \begin{aligned} \mathcal{J}_j(z;F) &= \frac{1}{(j-1)!} \int_0^z (z-x)^{j-1} dF(x) \\ &= \frac{1}{N(j-1)!} \sum_{i=1}^N \mathbf{1}\left(X_i \leq z\right) \cdot (z-X_i)^{j-1} \end{aligned} \]

Note that: \(\mathbf{1}\left(X_i \leq z\right)\) is the ECDF

SD and KS

  1. Hypotheses
    1. \(H_0^j\): \(\mathcal{J}_j(z;G) \leq \mathcal{J}_j(z;F) \qquad \forall z \in [0, \overline{z} ]\)
    2. \(H_a^j\): \(\mathcal{J}_j(z;G) > \mathcal{J}_j(z;F) \qquad \text{for some } z \in [0, \overline{z} ]\)
  2. Test statistic

\[ \hat{S}_j = \sqrt{\frac{NM}{N+M}} \left[ \sup_z \left( \mathcal{J}_j(z;\hat{G}_M) - \mathcal{J}_j(z;\hat{F}_N) \right) \right] \]

Rejection Rules

  1. Reject \(H_0\) if \(\hat{S}_j > c_j\)
  2. Seems easy, but it is less so in practice
  3. Need critical value, \(c_j\)
    1. For FSD, not too bad
    2. For SSD, TSD, not so nice

Basically, a KS-Test

  1. We can ‘easily’ calculate \(\hat{S}_1\), since it is a simple difference in cdf.
  2. The resulting critical values from \(c_1(\alpha) = \sqrt{ -0.5 \ln \alpha }\)
  3. Recall: \(\mathcal{J}_j(z;\hat{G}_M)\) is simply the ECDF over \(y\) data
  4. So, this seems relatively simple..
    1. Works in obvious cases
    2. Sometimes has wrong size
    3. This has been addressed in most software
  5. However, weighting of data is problematic.
